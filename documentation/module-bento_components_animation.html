bento/components/animation

Animation component. Draws an animated sprite on screen at the entity position.
Exports: Constructor
Parameters:
Name Type Description
settings Object Settings
Properties
Name Type Description
imageName String Asset name for the image. Calls Bento.assets.getImage() internally.
imageFromUrl String Load image from url asynchronously. (NOT RECOMMENDED, you should use imageName)
onLoad function Called when image is loaded through URL
frameCountX Number Number of animation frames horizontally (defaults to 1)
frameCountY Number Number of animation frames vertically (defaults to 1)
frameWidth Number Alternative for frameCountX, sets the width manually
frameHeight Number Alternative for frameCountY, sets the height manually
paddding Number Pixelsize between frames
animations Object Object literal defining animations, the object literal keys are the animation names
animations[...].loop Boolean Whether the animation should loop (defaults to true)
animations[...].backTo Number Loop back the animation to a certain frame (defaults to 0)
animations[...].speed Number Speed at which the animation is played. 1 is max speed (changes frame every tick). (defaults to 1)
animations[...].frames Array The frames that define the animation. The frames are counted starting from 0 (the top left)
Source:
  • components/animation.js, line 1
Returns:
Returns a component object to be attached to an entity.
Example
// Defines a 3 x 3 spritesheet with several animations
// Note: The default is automatically defined if no animations object is passed
var sprite = new Sprite({
        imageName: "mySpriteSheet",
        frameCountX: 3,
        frameCountY: 3,
        animations: {
            "default": {
                frames: [0]
            },
            "walk": {
                speed: 0.2,
                frames: [1, 2, 3, 4, 5, 6]
            },
            "jump": {
                speed: 0.2,
                frames: [7, 8]
            }
        }
     }),
    entity = new Entity({
        components: [sprite] // attach sprite to entity
                             // alternative to passing a components array is by calling entity.attach(sprite);
    });

// attach entity to game
Bento.objects.attach(entity);
