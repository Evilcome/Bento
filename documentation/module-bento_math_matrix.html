setValues(array) → {Matrix}

Set the values inside matrix using an array. If the matrix is 2x2 in size, then supplying an array with values [1, 2, 3, 4] will result in a matrix
[1 2]
[3 4]
If the array has more elements than the matrix, the rest of the array is ignored.
